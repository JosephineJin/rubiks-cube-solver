# 3×3×3 Rubik's Cube Solver

AI-based solver using IDA* with pattern database heuristics.

## Features
- Cubie model representation
- IDA* search with admissible heuristics
- Interactive visualization
- Sub-second solving for random scrambles

## Setup
pip install -r requirements.txt

## Usage
python demos/demo.py
